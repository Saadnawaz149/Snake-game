# Snake Game

Classic Snake built with a modern React + Vite setup.

## Live Demo
- https://snake-game-peach-five.vercel.app

## Tech Stack
- **Languages:** JavaScript (ES6), HTML, CSS
- **Frameworks/Libraries:** React 18
- **Tooling:** Vite, npm

## Concepts Covered
- Game loop with fixed tick rate
- Deterministic state transitions
- Collision detection (walls + self)
- Grid-based movement
- Randomized food spawning with occupied-cell checks
- Score tracking and win/lose states
- Keyboard input handling (Arrow keys + WASD)
- Responsive UI with touch controls (mobile)

## Features
- Classic Snake rules
- Growing snake on food pickup
- Game over + restart
- Pause/resume
- Mobile-friendly controls

## Controls
- Move: Arrow keys or WASD  
- Pause/Resume: Space or P  
- Restart: R
